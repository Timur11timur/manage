#About project

This is an open source forum

## Installation instructions:

* git clone https://github.com/Timur11timur/manage.git
* cd manage
* composer install
* npm install
* cp .env.example .env

Add necessary params to .env

* php artisan key:generate
* php artisan migrate
* php artisan db:seed
